Inspect SharePoint list columns

This is a PowerShell script (op-prem) I use a lot to inspect SharePoint list columns.

There are also great third party tools such as SPI-Insider and SharePoint Manager to do a deep-inspection of your SharePoint site. But I find that this simple PowerShell is also a great method to quickly inspect a list without starting a desktop application or browser plugin.

Preview

Usage

  • First parameter is the URL of the site
  • Second is the name of the list > If no list is found, all lists on the site will be displayed
  • Third (optional) list item ID, if no ID is supplied all list items in the list will be displayed
  • Fourth (optional) filter on the list column names

Code

 1param(
 2    [Parameter(Mandatory=$True,Position=1)] [string]$url,
 3    [Parameter(Mandatory=$True,Position=2)] [string]$listTitle,
 4    [Parameter(Mandatory=$False,Position=3)] [string]$listItemID,
 5    [Parameter(Mandatory=$False,Position=4)] [string]$filter
 6)
 7
 8Write-Host "usage: ListItems.ps1 [url] [listTitle] [listItemID] [filter]       (listItemID and filter is optional)" -ForegroundColor Yellow -BackgroundColor Green
 9if ($null -eq (Get-PSSnapin -Name Microsoft.SharePoint.PowerShell -ErrorAction SilentlyContinue))
10{
11    Add-PSSnapin Microsoft.SharePoint.PowerShell
12}
13
14try {
15    $web = Get-SPWeb $url -ErrorAction SilentlyContinue
16    $list = $web.Lists[$listTitle]
17} catch {
18    Write-Host "$url does not exist" -ForegroundColor Red
19    break
20}
21
22if(-not $listItemID)
23{
24    Write-Host "ListItemID was not supplied, listing all listitems" -BackgroundColor DarkGreen -ForegroundColor Black
25    if($null -eq $list)
26    {
27        Write-Host "No list found, showing all available lists" -ForegroundColor Red
28        $web.Lists | ForEach-Object {
29            Write-Host $_.Title
30        }
31    }
32    $list.Items | Format-Table
33}
34else 
35{
36    Write-Host "Showing specific ListItem: " -BackgroundColor DarkGreen -ForegroundColor Black
37    try {
38        $listItem = $list.GetItemByID($listItemID)
39    } catch {
40        Write-Host "Listitem $listItemID does not exists" -ForegroundColor Red
41        break
42    }
43    
44    if(-not $filter)
45    {
46        $listItem.Fields | ForEach-Object {
47            Write-Host $_.Title "[" $_.InternalName "]`t`t" -NoNewline -ForegroundColor Yellow -BackgroundColor DarkRed
48            Write-Host $listItem[$_.InternalName] -ForegroundColor Yellow -BackgroundColor Red
49        }
50    }
51    else
52    {
53        $listItem.Fields | Where-Object {$_.InternalName -match $filter} | ForEach-Object {
54            Write-Host $_.Title "[" $_.InternalName "]`t`t" -NoNewline -ForegroundColor Yellow -BackgroundColor DarkRed
55            Write-Host $listItem[$_.InternalName] -ForegroundColor Yellow -BackgroundColor Red
56        }
57    }
58}
